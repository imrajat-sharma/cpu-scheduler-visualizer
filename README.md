# CPU Scheduler

A project for simulating and analyzing CPU scheduling algorithms.

## Overview

This repository contains implementations of various CPU scheduling algorithms used in operating systems to manage process execution on a processor.

## Features

- Multiple scheduling algorithm implementations
- Process simulation and analysis
- Performance metrics and comparisons

## Getting Started

### Prerequisites

- HTML, CSS
- JavaScript 
- React
- Git

### Installation

```bash
git clone https://github.com/imrajat-sharma/cpu-scheduler.git
cd cpu-scheduler
```

## Usage

```bash
# Example usage
npm install
npm run dev
```

## Algorithms Supported

- First Come First Served (FCFS)
- Shortest Job First (SJF)
- Round Robin (RR)
- Priority Scheduling
